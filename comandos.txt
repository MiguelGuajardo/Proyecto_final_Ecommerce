Iniciar en modo FORK

node serverMongoDB.js -p 8080 -m FORK

Iniciar en modo CLUSTER

node serverMongoDB.js -p 8080 -m CLUSTER

EJECUCION DE SERVIDOR CON pm2

pm2 start serverMongoDb.js --name="server 1" --watch -- -p 8080 -m CLUSTER
pm2 start serverMongoDb.js --name="server 2" --watch -- -p 8082 -m FORK
pm2 start serverMongoDb.js --name="server 3" --watch -- -p 8083 -m FORK
pm2 start serverMongoDb.js --name="server 4" --watch -- -p 8084 -m FORK
pm2 start serverMongoDb.js --name="server 5" --watch -- -p 8085 -m FORK

ARTILLERY

artillery quick --count 50 -n 20 http://localhost:8080/lista > result_fork.txt

artillery quick --count 50 -n 20 http://localhost:8080/lista > result_cluster.txt
